# 宏仅仅是文本替换

  1. 使用括号避免错误
  2. 使用 do{}while(0) or ({...}) 定义多行宏，每行末尾添加续行符\
    1. do{}while(0)的好处是可以使用if等跳出语句
    2. 仅使用{}会造成麻烦--是否在结尾添加;--如在if()后使用，会使else错误
#define SQUARE(x) x * x
int tmp = SQUARE(3 + 3);  // tmp = 3 + 3 * 3 + 3

#define SQUARE(x) ((x) * (x))
int tmp = SQUARE(3 + 3);  // tmp = ((3 + 3) * (3 + 3))
